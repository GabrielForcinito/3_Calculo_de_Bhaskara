# 📊 Calculadora de Bhaskara em Python

## 📌 Sobre o projeto

Projeto desenvolvido em Python com o objetivo de calcular as raízes de uma equação do 2º grau (ax² + bx + c), utilizando a fórmula de Bhaskara.

O projeto apresenta diferentes abordagens, demonstrando a evolução na aplicação da lógica de programação.

## 🛠️ Tecnologias utilizadas

* Python

## 🎯 Funcionalidades

* Cálculo do delta (Δ)
* Verificação de raízes reais
* Cálculo das raízes da equação
* Diferentes versões do algoritmo (básico e aprimorado)

## 📝 Versões do projeto

### 🔹 Versão básica

* Implementação direta da fórmula
* Foco no aprendizado inicial

### 🔹 Versão aprimorada

* Melhor organização do código
* Uso de estruturas condicionais
* Tratamento de casos específicos (como delta negativo)
