# 📊 Calculadora de Bhaskara em Python

## 📌 Sobre o projeto

Projeto desenvolvido em Python com o objetivo de calcular as raízes de uma equação do 2º grau (ax² + bx + c), utilizando a fórmula de Bhaskara.

Este projeto aplica conceitos matemáticos e lógica de programação para resolver um problema clássico.

## 🛠️ Tecnologias utilizadas

* Python

## 🎯 Funcionalidades

* Cálculo do delta (Δ)
* Verificação de raízes reais
* Cálculo das duas raízes da equação
* Exibição dos resultados no terminal
