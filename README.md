# 📊 Calculadora de Bhaskara em Python

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de calcular as raízes de uma **equação do 2º grau (ax² + bx + c)**, utilizando a fórmula de Bhaskara.

O projeto apresenta diferentes abordagens, demonstrando a evolução na aplicação da lógica de programação e na organização do código.

## 🛠️ Tecnologias utilizadas

* Python

---

## 🎯 Funcionalidades

* Cálculo do delta (Δ)
* Verificação da existência de raízes reais
* Cálculo das raízes da equação
* Tratamento de diferentes casos (delta positivo, zero e negativo)
* Evolução do algoritmo com versões aprimoradas

---

## 📝 Versões do projeto

### 🔹 Versão básica

* Implementação direta da fórmula
* Foco no aprendizado inicial

### 🔹 Versão aprimorada

* Melhor organização do código
* Uso de estruturas condicionais
* Tratamento de casos específicos (como delta negativo)

## 💻 Como executar

1. Clone ou baixe este repositório
2. Abra o projeto no **Visual Studio Code**
3. Abra o terminal integrado
4. Execute o arquivo Python com o comando:

```bash id="j7e3yh"
python nome_do_arquivo.py
```

> 💡 Substitua `nome_do_arquivo.py` pelo arquivo que deseja executar.
> 💡 Recomenda-se a execução pelo terminal do VS Code para melhor visualização e interação com o programa.
